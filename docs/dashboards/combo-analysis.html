Кейс Starbucks 2024-2026

В июне 2024 Starbucks ввёл breakfast combos $5-6 (кофе + круассан/сэндвич) в ответ на:
  • −7% транзакций
  • −3% comp.sales США
Результат: объём заказов восстановился к Q4 2024. Комбо удерживают трафик в момент когда «всё дорого».

Для Bounty это же — превентивная мера: пока конкуренты не сделали комбо, занять нишу.

★ План внедрения

7 шагов на 2 недели

  • 1Запросить дегустацию у 3 кейтеринг-поставщиков (Bankatering, TakeEat + 1 локальный). Образцы 7 видов сэндвичей1-2 дня
  • 2Выбрать основного поставщика по вкусу+цене+логистике. Подписать рамочный договор на 1 месяц теста3-4 дня
  • 3Bounty-design добавляет комбо в меню (3 стартовых: Утренний, Классика, Бизнес-ланч). Печать обновления5-6 дней
  • 4Закупить холодильник-витрину (если нет) — $80-120 один раз. Или использовать существующую5-7 дней
  • 5Обучение бариста (1 день): как предлагать комбо, как избегать «навязывания», когда говорить «не хотите комбо?»7 день
  • 6Запуск только на Ц-14 (где трафик больше, риск ниже). bounty-smm делает анонс в IG: 3 Reels + Stories7-14 день
  • 7Стив + bounty-pricing аудит по итогам 2 недель: % конверсии, маржа, списания. Решение масштабировать на Сайрам или нет21 день

★ Итог

Резюме для решения

✓ Внедрять. Прогноз доп.выручки +20-30M сум/мес на 2 точки в базовом сценарии. Маржа комбо 47-64% (healthy).

✓ Стартовать с 3 комбо (не 5+), на 2 недели, только Ц-14. Расширение — по результату.

✓ Поставщик: Bankatering или TakeEat. Дегустация у 3 кандидатов на этой неделе.

⚠ Стив должен сверить с НК РУз режим продажи готовой еды через 3rd-party + лицензия. bounty-risk — оценка регуляторных рисков перед стартом.

📊 KPI первые 2 недели: конверсия в комбо ≥15%, списания ≤15%, маржа ≥45%, средний чек +10%. Если все 4 — масштабируем на Сайрам.
