## 1. Палитра

Извлечена визуально из меню. **HEX-значения приблизительные**, требуется уточнение через инспектор цвета в исходниках бренда (Adobe Color/Pipette). Сейчас отдаю best-estimate.

### Основные (3 главных цвета)

| Роль | Название | HEX (≈) | RGB (≈) | Где используется |
|---|---|---|---|---|
| Primary / фон | Тёмно-синий «полночный» | `#0F1F3A` | 15, 31, 58 | Полная заливка фона меню |
| Accent / премиум | Золото / горчично-жёлтый | `#C9A14A` | 201, 161, 74 | Логотип, пальмовые листья, цена-плашки, заголовки разделов |
| Контраст / текст | Off-white | `#F2EAD7` | 242, 234, 215 | Названия позиций, основной текст |

### Вспомогательные / служебные

| Роль | HEX (≈) | Применение |
|---|---|---|
| Полупрозрачный белый | `rgba(255,255,255,0.12)` | Разделительные линии между категориями |
| Светло-голубой/сталь | `#7A8FA8` | Подписи к колонкам объёмов (250 МЛ / 500 МЛ) |

### Колор-логика

- **Контраст синий+золото** — устойчивая премиальная пара, читается как «luxury hospitality». Уход в эту сторону = ресторан/отель, а не повседневная кофейня.
- Off-white вместо чистого `#FFFFFF` — мягкий, ассоциируется с «бумагой / винтажем».
- **Внимание (риск):** в Ташкенте «синий+золото» как пара уже эксплуатируется ресторанами высокого сегмента — Bounty может потеряться. Стоит обдумать, **усилить ли отличие** через добавление 4-го цвета (например, бирюзовый/морской — поддержит «тропическую» тему пальм).

## 2. Шрифты

Точные имена начертаний без исходного файла не определить — даю описание по форме.

### Заголовок логотипа «BOUNTY»

- Тип: **Serif, high-contrast, didone-подобный**
- Характеристики: тонкие штрихи, заглавные буквы, широкий трекинг
- Очень похоже на **Bodoni / Didot** или аналог (Playfair Display)
- Назначение: люкс-сегмент, fashion-эстетика

### Подзаголовок «Coffee bar»

- Тип: **Script / handwritten italic**
- Тонкое, ручное, контрастирует с строгим Bounty
- Похоже на **Cookie / Allura / Dancing Script**

### Заголовки разделов («КОФЕ», «КОФЕЙНЫЕ НАПИТКИ», «НАПИТКИ», «ДОБАВКИ»)

- Тип: **Sans-serif, geometric, bold, all-caps, широкий трекинг**
- Похоже на **Futura / Montserrat / Gotham (bold)**

### Тело меню (названия позиций, цены)

- Тип: **Sans-serif, regular/medium, all-caps**
- Чистая гротеска, экономная
- Похоже на **Roboto Condensed / Inter / Helvetica Neue**

### Подписи объёмов (250 МЛ / 500 МЛ)

- Тип: **Sans-serif, light, маленький кегль**

### Системная диагностика

Меню использует **4 разных гарнитуры** (didone + script + geometric sans bold + sans regular). Это работает на печатной точке, но **в Instagram-формате это много** — нужно зафиксировать иерархию из 2 (макс 3) шрифтов.

**Рекомендация системы:**
- Display: didone-serif (логотип, заголовки артикулов)
- Body / UI: geometric sans (Montserrat / Inter)
- Script — **только в логотипе**, не использовать в постах

## 3. Декоративные элементы

| Элемент | Описание | Семантика |
|---|---|---|
| **Логотип** | Круглая печать тёмно-синего цвета с золотой пальмой по центру и надписью «BOUNTY / Coffee bar» | Премиум + тропики |
| **Золотые пальмовые листья** | Симметрично в углах шапки, рисованные, листья кокосовой/финиковой пальмы, тонкая линия | Тропики / отпуск / Карибы / батончик Bounty (рефренс) |
| **Цены-плашки** | Золотые горизонтальные полосы под заголовками «(250 МЛ / 500 МЛ)» | Структурный разделитель |
| **Размерные стрелки** | Голубые двойные стрелки по краям с указанием физических размеров (450 мм × 700 мм) | Это **техническая разметка препресса**, не должна быть на финальном меню в зале. Возможный артефакт макета. |

### Семантический конфликт

Логотип уводит в **«остров / тропики / батончик Bounty»** (пальмы, кокос).
Цвет-палитра уводит в **«отель / luxury»** (синий+золото).
Меню по сути — **городская кофейня на районе** (Ц-14, Хадра).

**Это три разных сигнала**, и они не складываются в один read. Нужно решение на уровне бренд-стратегии:
- Усилить тропики → добавить морской/бирюзовый, упростить шрифт, ввести иллюстративность
- Усилить luxury → убрать пальмы, заменить на минималистичный coffee-mark, оставить чистый didone
- Найти микс → определить **главный сигнал** (вероятно «escape / маленький отпуск в чашке») и подчинить остальное

## 4. Композиция текущего меню

### Структура (сверху вниз)

1. **Header**: пальмовые листья (углы) + круглый логотип по центру
2. **Раздел КОФЕ**: 2 колонки (чёрный кофе + молочный кофе), таблица «250/500 мл»
3. **Раздел КОФЕЙНЫЕ НАПИТКИ**: 2 колонки (горячие × холодные), «250/500 мл»
4. **Раздел НАПИТКИ**: 2 колонки (горячие × холодные), «500 мл / 250/500 мл»
5. **Раздел ДОБАВКИ**: 2 колонки (добавки × декор)
6. **Footer**: только размерные стрелки (артефакт препресса)

### Что хорошо

- Логичная иерархия от «черный кофе» (база) к «добавкам» (хвост).
- Чёткий ритм «заголовок → жёлтая плашка с объёмами → таблица позиций».
- Симметричная двухколоночная организация даёт визуальный баланс.

### Что улучшать (для постера/IG-адаптации)

- Нет **визуального якоря** в каждом разделе (нет иконок/иллюстраций напитков).
- Все позиции выглядят равновесно — нет «hero / signature drink», которого хочется заказать.
- Цены сливаются с названиями (одинаковый кегль) — нет точки фокуса.
- Меню перегружено для онлайн-формата: 30+ позиций. Для IG-сториз нужна **редактированная версия** на 6–9 hero-позиций.
