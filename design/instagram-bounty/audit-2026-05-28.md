## Чего НЕТ (нужны скриншоты от владельца)

Ниже — конкретный чек-лист. Каждый пункт обязателен для нормального аудита.

### Профиль (1 файл)
- [ ] Скриншот шапки профиля во всю ширину: аватарка, имя, био, ссылка, кнопки, highlight-кружки (с подписями)
- [ ] Скриншот сетки последних **12 постов** (3×4) — нужна композиция, цвет, ритм
- [ ] Скриншот раздела «Закреплённые» (3 поста, если есть)
- [ ] Скриншот раздела Reels (топ-9 по охвату)
- [ ] Скриншот раздела Метки/Tagged

### Insights (Professional dashboard) — нужны цифры
- [ ] Insights → Обзор за **последние 30 дней**: охват, показы, посещения профиля, клики по ссылке
- [ ] Insights → Контент → топ-9 постов по охвату за 30 дней
- [ ] Insights → Аудитория: возраст, пол, города, активность по часам/дням
- [ ] Insights → Сториз за **последние 7 дней**: охват каждой, exits, replies, taps forward/back
- [ ] Insights → Reels: охват, plays, watch-through rate

### Контент-семплы
- [ ] **Один Reel в hi-res** (любой свежий) — выгрузить через «Сохранить» в галерею и прислать MP4
- [ ] 3 фото-поста разных типов: продукт (стакан), интерьер, людское/lifestyle
- [ ] 5 свежих сториз подряд (любая активная подборка)

### Highlight-обложки
- [ ] Скриншот всех highlight-обложек крупно (одной полосой)

### Конкуренты, на которых смотрит владелец
- [ ] Список 3–5 IG-аккаунтов, которые владелец считает референсом
