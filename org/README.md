## Потоки данных (кто кому что передаёт)

- **Прайс-аналитик** → **Стив** (себестоимость для отчётов) + **Маркетолог** (рамки для акций)
- **Конкурент-интел** → **Маркетолог** (на что реагировать) + **Локация-аналитик** (где у конкурентов точки)
- **Локация-аналитик** → **Риск-менеджер** (оценка рисков точки) + **Стив** (CapEx-модель)
- **Операц.менеджер** → **Стив** (расходы) + **HR-функция в нём же** (фонд оплаты бариста)
- **SMM** → **Дизайнер** (брифы на визуал) + **Маркетолог** (кампании)
- **Все** → **Я (Chief of Staff)** → **Тебе** (финальные решения)

## Правила работы

1. **Решения принимаешь только ты.** Агенты не самостоятельны — они анализируют и рекомендуют.
2. **Все артефакты — в свою папку.** Никто не пишет вне своей.
3. **Перед финансовым ответом — через Стива.** Это правило из memory.
4. **Все цифры с источником.** Excel-книга, конкретный URL, дата скрейпа.
5. **Эскалация.** Если агент сомневается на 8/10 — он стопает и спрашивает тебя через меня.
